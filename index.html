Description and Rationale

Our group have decided to make use of several modern web technologies and techniques to further improve the user experience of the current COMP 1950 website.

Technologies Used

Bootstrap Framework
Leverage the framework's ability to display responsive web pages and maintain consistency through the use of Themes & Templates. We've chosen SB Admin 2 as the foundation for the new COMP 1950 website.
Jquery
Performs user input validation with dynamic data delivery and retrieval (JSON) which allows students to take their quizzes online.

Techniques Applied:

Security
Implement client-side (JavaScript) and server-side (httpd.conf or .htaccess) configuration when handling user information and user friendly error messages (i.e "404 page not found").
Search Engine Optimization
Applied Google's SEO Guidelines and implement semantically structured web pages. Quizzes section are configured to turn away search engine bots to avoid appearing on online search results.
Presentation
Made use of a stylesheet reset and a separate printer friendly stylesheet.
User Experience
Includes familiar buttons and icons, notification displays, chat support, and accessibility features.
